# 🦐 三虾协作日志系统

## 📁 目录结构

```
logs/three-shrimp/
├── README.md              # 本说明文档
├── STANDARD.md            # 📋 三虾协作标准规范（必读）
├── YYYY-MM-DD.md          # 📄 每日主日志（一天只有一个！）
└── archive/               # 归档目录
    └── 2026-03/
        └── 2026-03-01.md  # 历史日志按月归档
```

## 🎯 每日主文档（重要）

**每一天只有一个主文档，所有协作内容都记录在这里：**

| 项目 | 说明 |
|------|------|
| **文件名** | `YYYY-MM-DD.md`（如 `2026-03-24.md`） |
| **位置** | `logs/three-shrimp/` 根目录 |
| **创建者** | 来福虾（当日首次协作时创建） |
| **维护者** | 来福虾（实时更新） |
| **内容** | 任务清单、对话记录、待办事项 |

### ❌ 错误示范
```
logs/three-shrimp/
├── 2026-03-24-来福虾.md    # ❌ 不要按虾分文件
├── 2026-03-24-扣子虾.md    # ❌ 一天只有一个文件
└── 2026-03-24-Q福虾.md     # ❌ 所有内容写进主文档
```

### ✅ 正确示范
```
logs/three-shrimp/
└── 2026-03-24.md           # ✅ 一天只有一个主文档
    # 内部按章节记录各虾的活动
```

## 📝 日志命名规范

- **主文档格式**: `YYYY-MM-DD.md`
- **示例**: `2026-03-24.md`
- **创建时机**: 当日首次有协作任务时由来福虾自动生成
- **唯一性**: 一天只有一个主文档，所有内容都写在这里

## 📋 日志内容模板

每篇日志包含以下章节：

1. **📋 今日概览** - 日期、天气、在线状态、任务统计
2. **🎯 今日任务清单** - 任务详情、执行虾、状态
3. **💬 关键对话记录** - 重要沟通内容摘要
4. **🔧 系统优化** - 协作流程改进记录
5. **📝 备注与待办** - 遗留问题、改进建议
6. **🏷️ 标签** - 便于检索的关键词

## 🚀 快速开始

### 查看今日日志
```bash
# 查看最新日志
cat logs/three-shrimp/$(date +%Y-%m-%d).md
```

### 查看历史日志
```bash
# 查看昨天
cat logs/three-shrimp/$(date -d "yesterday" +%Y-%m-%d).md

# 查看指定日期
cat logs/three-shrimp/2026-03-24.md
```

### 搜索历史记录
```bash
# 搜索包含特定关键词的日志
grep -r "天气测试" logs/three-shrimp/

# 搜索特定标签
grep -r "#系统优化" logs/three-shrimp/
```

## 🔄 自动化机制

1. **每日自动创建**: 来福虾在当日首次接收到协作任务时，自动创建当日日志
2. **实时追加**: 任务完成或重要沟通后，实时追加到当日日志
3. **定期归档**: 建议每月将旧日志归档到 `logs/three-shrimp/archive/`

## 👥 角色职责

| 角色 | 日志职责 |
|------|----------|
| **来福虾** | 统筹记录、日志创建与维护 |
| **Q福虾** | 本地任务执行记录、数据更新 |
| **扣子虾** | 可视化/代码任务记录、成果归档 |

## 📌 使用建议

1. **每日回顾**: 工作日结束时快速浏览当日日志，确认无遗漏
2. **问题追溯**: 遇到协作问题时，按日期检索相关记录
3. **经验沉淀**: 定期整理日志中的优秀实践，更新到 MEMORY.md
4. **月度总结**: 每月末汇总当月日志，形成协作报告

## 🔍 检索技巧

### 按日期检索
```bash
# 查看某月所有日志
ls logs/three-shrimp/2026-03-*.md
```

### 按任务类型检索
```bash
# 查找所有PPT相关任务
grep -l "PPT\|幻灯片" logs/three-shrimp/*.md
```

### 按执行虾检索
```bash
# 查找Q福虾参与的任务
grep -l "Q福虾" logs/three-shrimp/*.md
```
