### 核心内容摘要

**文献信息**:
- 标题: 工业智能体：体系架构、关键技术与未来展望
- 作者: 任磊、王海腾、董家宝等（北航）
- 发表: 《中国科学：技术科学》2026年
- 定位: 国内首篇系统化阐述工业智能体的综述论文

**核心定义**:
工业智能体 = 大模型/具身智能 + 工业产品全生命周期 + 具身/非具身形态

**五大特征**（vs传统智能体）:
1. 规则驱动 → 大模型驱动
2. 被动响应 → 主动规划决策
3. 离线学习 → 自适应演化
4. 短期记忆 → 长期记忆
5. 单体智能 → 多智能体自主协同

**三层架构**:
- 应用层: 研-产-试-管-服全流程
- 系统层: 软件数据流+硬件控制流+人类认知流
- 基座层: 感知、记忆、规划、通信、执行、演化学习

**四大挑战**:
1. 动态自适应（数据漂移、泛化不足）
2. 可靠性保障（机理融合、透明度）
3. 多智能体协作（异质性、通信瓶颈）
4. 长程指令执行（长上下文理解）

**技术路线**: 预训练 → 指令微调 → 强化学习

**核心洞察**:
从"刚性自动化"向"高自律、强泛化、自主进化"的新型工业化形态迈进
