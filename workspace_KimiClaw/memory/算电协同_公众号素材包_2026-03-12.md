## 📌 一句话定位

**算电协同** = 算力与电力在规划、建设、运营全环节深度融合，从"算力等电"转向"算电一体"。

### 一次大模型训练的耗电
- **千亿参数大模型训练**：耗电量可达**1200万度**
- 相当于：**5万人口小镇全年用电量**

### 成本结构
- 数据中心运营成本中，**电力+算力折旧占比70%**
- 电费占算力运营成本 **60%-70%**
- 度电成本每降低0.1元，大型数据中心可带来**数亿元级利润差异**

## 💡 核心逻辑

### 双向协同
```
算力跟着电力走 → 高负载AI训练向西部转移（就地消纳廉价绿电）
电力跟着算力建 → 东部保障实时AI应用（就近供电）
```

### 战略意义
1. **解决空间错位**：西部有电没算力，东部有算力没电
2. **降低成本**：用0.2元的绿电跑算力，vs 东部0.6元+的工业电
3. **绿电消纳**：解决西部风光电"发得出、送不出、存不住"
4. **Token出海**：电不出境，价值已远航（算力服务出口）
