> 本文件记录多虾协作聊天日志的存储规则，供所有虾参考。

---

## 存储位置

GitHub 仓库: `skygentwu/miniopc-write-report/chatlog_Agents/`

本地路径: `~/.openclaw/workspace/chatlog_Agents/`

---

## 文件命名规范

格式: `duoxia-YYYYMMDD.md`

- **duoxia**: 多虾（固定前缀）
- **YYYYMMDD**: 年月日，如 20260326
- **每天一个独立文件**

示例:
- duoxia-20260326.md
- duoxia-20260327.md
- duoxia-20260328.md

---

## 记录内容

以下内容应归档到当日日志文件中：

1. **虾队状态检查报告** - 各虾的能力自检、状态评估
2. **群聊讨论记录** - 虾之间的协作对话
3. **工作流优化建议** - 效率改进、工具推荐
4. **协作问题和解决方案** - 遇到的问题及处理方法
5. **PM指令和任务分配** - 来自PM的明确指示
