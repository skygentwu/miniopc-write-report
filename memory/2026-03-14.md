### 核心定义
**工业智能体**：以大模型、具身智能等新一代AI技术驱动的，服务于工业产品全生命周期应用的新型智能体系统。

### 五大关键特征（vs 传统智能体）
1. 规则驱动 → 大模型驱动
2. 被动响应 → 主动规划决策
3. 离线学习 → 自适应演化
4. 短期记忆 → 长期记忆
5. 单体智能 → 多智能体自主协同

### 三层架构
- **基座层**：认知计算内核（感知、记忆、规划、通信、执行、学习）
- **系统层**：数字-物理接口中枢（软件数据流+硬件控制流+人类认知流）
- **应用层**：研-产-试-管-服全流程应用

### 四大挑战
1. 工业动态环境的自适应问题
2. 工业自主决策的可靠性问题
3. 工业多智能体的自组织协作问题
4. 复杂长程指令的理解与执行问题

### 关键技术
- 基座技术：预训练 → 指令微调 → 强化学习
- 功能模块：感知、记忆、规划、通信、执行、演化学习
- 系统编排：多智能体协同、任务分解、冲突仲裁

### 核心洞察
工业智能体的本质是推动工业体系从"刚性自动化"向"高自律、强泛化、自主进化"的新型工业化形态迈进。
